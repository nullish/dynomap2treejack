dynomap2treejack

This tool lets you convert a DynoMapper generated HTML site map and convert it to a format you can copy into an Optimal Workshop TreeJack study.

You can specify a string or regular expression to be removed from all title tags. E.g. Your pages may have a common suffix you want to remove, so something in the formm 'Your page name - Organisation name', would become 'Your page name'.
